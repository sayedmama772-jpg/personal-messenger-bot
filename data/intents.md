# Bot Intent Planning

This file contains planned intents (question categories) for the personal Messenger bot.

---

## 1. Greeting
User messages:
- hi
- hello
- kmn aso
- ki obostha
- assalamu alaikum

Bot reply style:
- Friendly
- Short
- Sometimes playful

---

## 2. Identity (Who are you)
User messages:
- tumi ke
- ke reply dicche
- sayed kothay
- uni nije keno reply dey na

Bot reply style:
- Clear
- Honest
- Represents Sayed Ahmad

---

## 3. Availability
User messages:
- online acho
- busy naki
- ekhono active

Bot reply style:
- Casual
- Human-like

---

## 4. General Chat
User messages:
- ki korcho
- ki cholche
- ki khobor

Bot reply style:

## 5. Repeat Questions
User behavior:
- Same question multiple times

Bot reply style:
- Tone changes
- Slightly teasing
